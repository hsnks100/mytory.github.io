간단하다. 터미널에 쳐 주자.

<pre class="brush:plain">sudo apt-get install kubuntu-desktop</pre>

넷북 에디션을 사용할 거면 

<pre class="brush:plain">sudo apt-get install kubuntu-netbook</pre>

이라고 쓰면 된다.
