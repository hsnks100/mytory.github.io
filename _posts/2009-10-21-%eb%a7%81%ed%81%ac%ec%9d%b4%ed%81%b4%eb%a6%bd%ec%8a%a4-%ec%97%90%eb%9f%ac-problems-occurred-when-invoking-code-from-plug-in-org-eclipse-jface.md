이클립스 에러: Problems occurred when invoking code from plug-in: &#8220;org.eclipse.jface&#8221;.

핵심만 말하면

> 이클립스 실행파일의 바로가기를 생성 한 후 마우스 오른쪽 버튼을 눌러서 속성 선택..
> 
> 대상에서 끝에 -clean을 입력 (플러그인 설정을 초기화하는 것이라고 하는 군요)

위처럼 하면 해결된다.
